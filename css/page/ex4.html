CODE CSS

  gap: 15px;
}

.card {
  background: white;
  padding: 20px;
  text-align: center;

4.2 Layout complet avec Grid Areas

CODE CSS
.page-layout {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
  grid-template-columns: 200px 1fr;
  gap: 10px;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }
