Correction TP CSS

1. Sélecteurs CSS

1. Sélectionnez tous les liens dans la navigation

.main-nav a {
}

2. Sélectionnez uniquement le lien actif

.main-nav a.active {
}

3. Sélectionnez l'article qui a la classe featured

article.featured {
}

4. Sélectionnez le premier paragraphe de chaque article

article p:first-of-type {
}

5. Sélectionnez les éléments <li> pairs

li:nth-child(even) {
}

1.2 Spécificité

1. Classez ces sélecteurs par spécificité croissante

Ordre (du plus faible au plus fort) :
  • 1. p
  • 2. .text
  • 3. p.text
  • 4. #main
  • 5. #main .text p

2. Si deux règles ont la même spécificité, laquelle gagne ?

C'est la règle qui est écrite en dernier dans le fichier CSS (la plus bas dans le code) qui l'emporte.

3. Pourquoi faut-il éviter !important ?

Il faut l'éviter car cela rend le débogage très difficile et casse la cascade naturelle du CSS
