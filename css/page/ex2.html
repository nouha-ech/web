Exercice 2

2.1 Comprendre le Box Model

1. Avec box-sizing: content-box (défaut), quelle est la largeur totale visible ?

350px
Calcul : width (300) + padding-left (20) + padding-right (20) + border-left (5) + border-right (5) = 350px.

2. Avec box-sizing: border-box, quelle est la largeur totale visible ?

300px
Avec border-box, le padding et la border sont inclus dans la largeur définie.

3. Écrivez le reset CSS universel pour box-sizing

*, *::before, *::after {
    box-sizing: border-box;
}

2.2 Centrer un élément

Centrez horizontalement et verticalement une div de 600px

HTML : 
<body>
    <div class="centered-box">Contenu</div>
</body>
CSS : 
body {
    display: flex;
    justify-content: center;
    align-items: center;
    min-height: 100vh;
    margin: 0;
}

.centered-box {
    width: 600px;
}
